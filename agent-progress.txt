## Server Actions for Notes - 2026-03-26

Created `lib/actions/notes.ts` with all four server actions:
- `createNoteAction()` — creates note with defaults, redirects if unauthenticated
- `updateNoteAction(noteId, data)` — validates with Zod, enforces ownership, revalidates path
- `deleteNoteAction(noteId)` — deletes note, revalidates dashboard
- `toggleShareAction(noteId, isPublic)` — calls setNotePublicSharing, returns publicSlug

All actions use `getSession()` for auth and enforce user ownership via repository functions.

Next recommended steps: CreateNoteButton component, dashboard loading/error states, E2E tests.
